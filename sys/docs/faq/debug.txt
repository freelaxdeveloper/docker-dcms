[title]Режим разработчика[/title]
Режим разработчика включает отображение ошибок интерпретатора PHP для создателя сайта.
Эта функция помогает разобраться в причинах неработоспособности отдельных модулей или тем оформления.
[b]![/b] Отключение данной опции не отключит отображение XML ошибок в темах оформления.
Также данная опция подволяет изучить места, которые вызывают увеличение времени генерации страницы. Для этого нужно вызвать страницу с GET параметром [b]debug[/b].
В код страницы добавится информация с участками кода и временем их выполнения.
Свои участки кода можно добавить с помощью статического метода debug::step('описание');
